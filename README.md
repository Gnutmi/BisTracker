# WoW Midnight — Season 1 Raid BiS Reference

An interactive Best-in-Slot gear reference for all 40 specializations in World of Warcraft: Midnight (patch 12.0.1, Season 1).

**[View Live Tool](https://gnutmi.github.io/BisTracker/midnight-bis-reference.html)**

---

## Features

- All 40 specs across 13 classes (including the new Devourer Demon Hunter)
- Tier set pieces with 2pc/4pc bonus text and which slot to skip
- BiS weapons and trinkets with Wowhead tooltip support (hover any item link)
- Filter by class, role (Tank / Healer / Melee DPS / Ranged DPS), or free-text search
- Click any spec row to expand full gear details
- Dark void-themed UI with WoW class colors

## Usage

No build tools or dependencies required. Just open `midnight-bis-reference.html` in any browser, or use the live link above.

## Data Sources

Compiled from Wowhead Beta, Method.gg, Maxroll.gg, and Icy Veins (March 2026).
